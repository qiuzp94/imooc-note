## 题目
自己来Mock一个数据结构的接口，结构如下

```
    "Status":0,
    "data":[
        {
            "tid":"1",
            "title":12154545,
            "catalog":"index",
            "created":"1436864169",
            "isEnd":"0",
            "answer":"10",
            "user":{
                "avatar":"用户头像",
                "name":"用户昵称"
                “isVip”:"1",
                "level":"4"
            }
        }
    ]
```

### 要求与建议
1、了解数字、日期、字符串的随机数据的产生方式
2、如何发送简单的数据请求，并让Mock使用正则匹配lists请求。
3、获取传入的参数，并生成指定数量的Mock测试数据
4、可以使用start.zip中的代码，但是，需要自己先思考下。
