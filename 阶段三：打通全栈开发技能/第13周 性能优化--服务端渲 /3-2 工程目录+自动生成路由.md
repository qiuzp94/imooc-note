## Nuxt.js 基础
### 一、Vue应用工程目录
* src
    * componets 存放业务组件
    * views 页面视图
    * api 页面请求
    * router 路由
    * utils 公共配置
    * store vuex

### 二、Nuxt应用工程目录
* layouts 视图
* pages 页面、用于形成路由
* componets 存放业务组件
* assets 预编译资源sass
* plugins 插件配置
* middleware 模块
* static 静态资源(roboots.txt等)
* store vuex
